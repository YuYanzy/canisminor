# 沪江小 D

沪江小 D 词典(dict.hujiang.com)是沪江(hujiang.com)旗下国内领先的多语种词典工具。沪江小 D 词典为用户提供专业的查词服务，拥有强大的离线词库，无需网络，即可便捷查词。依托沪江网丰富的内容学习资源，沪江小 D 词典提供更多的拓展服务，助您学习路一臂之力。
